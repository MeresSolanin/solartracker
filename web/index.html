solartracker

private BitTorrent tracker generation

solartracker lets you share torrent files just with the people you invite to your tracker. Unlike public trackers, it shares peers only within a group of which all members use the same Announce URL. It really works like a private tracker, but it can be generated with the click of a button.

Your announce URL:

http://solartracker.rf.gd/{{room}}/announce

Soooooooooo.. how do I create a private torrent?

Using transmission on Linux:

  • File → New…
  • Select file to share
  • In Trackers field enter http://solartracker.rf.gd/{{room}}/announce
  • Select Private torrent
  • Click New
  • Click Add in next window
  • Click Open to start seeding
  • Oh yeah! Now you can send all those private torrents to your friends!
