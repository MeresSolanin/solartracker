solartracker

private BitTorrent tracker generation

solartracker allows to share torrent files just with your friends, nobody else. Unlike public trackers, it shares peers only within a group which is using the same Announce URL. It's basically just a private tracker, but can be generated with one click.

Your announce URL:

http://solartracker.rf.gd/{{room}}/announce

Sooooooo, how do I create a private torrent now?

Using transmission on Linux:

  • File → New…
  • Select file to share
  • In Trackers field enter http://solartracker.rf.gd/{{room}}/announce
  • Select Private torrent
  • Click New
  • Click Add in next window
  • Click Open to start seeding
  • Done. Now you can send that torrent file to all your friends.…
